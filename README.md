# PyGather News Aggregator

This project is a Python news aggregator that collects headlines from online news websites and displays them in a graphical interface.

## Features
- Scrapes latest news headlines
- Displays them in a desktop GUI
- Refresh button to update news

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Tkinter

## How to Run

Install required libraries:
pip install -r requirements.txt

Run the program:
python main.py
